# 場景圖片資料夾

將你的場景圖片放在這個資料夾中。

## 使用方式

1. 將圖片檔案放入此資料夾（支援 .jpg, .png, .gif 等格式）
2. 在 `data/story.json` 中定義場景，並指向對應圖片
3. 設定觸發該場景的關鍵字

## 範例

假設你有一張圖片 `forest.jpg`：

1. 將 `forest.jpg` 放入此資料夾
2. 在 `data/story.json` 中添加：

```json
{
  "scenes": [
    {
      "id": "forest",
      "name": "森林",
      "keywords": ["森林", "樹木", "叢林", "綠意"],
      "image": "images/forest.jpg",
      "description": "茂密的森林場景"
    }
  ]
}
```

3. 當 AI 生成的故事中出現「森林」、「樹木」等關鍵字時，圖片就會自動顯示

## 圖片建議

- **解析度**：建議 1920x1080 或 16:9 比例
- **檔案大小**：建議小於 1MB（以加快載入速度）
- **格式**：PNG 或 JPG
- **命名**：使用英文和底線，如 `dark_castle.jpg`

## 圖片來源建議

你可以從以下途徑獲取圖片：

- 自己繪製或拍攝
- 使用 AI 圖片生成工具（Stable Diffusion、DALL-E、Midjourney 等）
- 免費圖庫（Unsplash、Pexels、Pixabay 等）
- 確保圖片使用權限符合你的需求

## 注意事項

- 確保圖片路徑正確（區分大小寫）
- 如果圖片無法顯示，檢查瀏覽器控制台的錯誤訊息
- 可以為同一場景準備多張圖片，隨機或按條件顯示
